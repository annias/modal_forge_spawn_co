Control Deck Modules

A complete reference guide of all 32 ctl-modules that govern the Modal Forge generation cycles.

Infra

ctl-deck-shell

Client-side assembler that lays out the whole Control Deck panel UI.

Infra

ctl-mi-shell

Sub-panel assembler for the Motion Intelligence dial cluster.

Infra

ctl-forge-bridge

Bridges deck state parameters into the forge request.

Infra

ctl-brief-compose

Server-side aggregator: turns locked dials into the constraint block the generator reads.

Infra

ctl-mi-compose

Coherence engine that keeps the Motion Intelligence dials harmonized with each other.

Infra

ctl-presets

The 34-preset library: apply a preset to snap every dial, save your own custom presets.

Infra

ctl-summary-chip

Display-only readout pill showing how many dials are currently locked.

Column structure (single / two-column key-value) and grouped vs. flat sections.

Layout

Locks modal width + optional snap grid.

Header region: icon style, title, subtitle/breadcrumb/meta, close-button placement.

Footer style (gradient / solid / ghost) and alignment.

Accent color (cyan / magenta / terminal-green / orange / violet).

Forge Director Refinement

The original generation system has been augmented with the powerful Forge Director flow to facilitate advanced visual tweaks:

  • Hybridize takes: Breed traits from two selected modals to generate a blended child card.
  • Mutate styles: Take a single favorite layout and spin off 4 sub-variations.
  • palette swatching: Input visual palettes directly or let the auto-palette balance colors.
🧠

Motion Intelligence (Feel Layer)

Configure how interfaces *feel* during loading and user interaction, not just how they look statically:

  • Emotional Tones: Sync pacing with Calm, Playful, Aggressive, or Mysterious themes.
  • Signature Moves: Toggle custom shaders: CRT screen flickers, holographic drift, or trails.
  • Layer coherence: Ensures hover animations coordinate with entrance duration curves.
